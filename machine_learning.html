Intro ------------------------------

Machine Learning (ML) grew out of work in AI (Artificial Intelligence).
Examples:
  - database mining (web click data, medical logs, biology, engineering)
  - Hand-writing recognition, autonomous helicopter, NLP (natural language processing), computer vision (recognizing images)
  - self-customizing programs (e.g. Amazon , Netflix recommender engines).
  - understanding human learning (brain, real AI)

What is Machine Learning?

"Machine Leaning is a field of study that gives computers the ability to learn without being explicitly programmed." - Arthur Samuel (1959) - programming checkers play.

"Well-posed Leaning Problem: A computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured by P, improves with experience E." - Tom Mitchell (1998)

Machine Learning algorithms: Supervised and unsupervised, reinforcement learning, recommender systems.

You need tools to develop learning algorithms, and you need to know how to use these tools (best practicies).

Supervised learning - we have sets of correct answers, and then we use regression to make predictions of value. Or classification problems (predict classification result). Example - predict type of tumor based on several parameters (for example, size and age, etc.). For some learning problems we have infinite number of attributes.

Unsupervised learning - clusterring, finding structure, recommend structure, market segmentation, analysis of astronomical data, etc. Example - coctail party problem (separating voices of speakers, separating voice form music, etc.).
