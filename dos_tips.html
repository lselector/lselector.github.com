DOS Tips

===============================================
To activate autocomplete:
  run regedit
  find  Command Processor:
  MyComputer \ HKEY_CURRENT_USER \ Software \ Microsoft \ Command Processor

The CompletionChar should be changed to 9 (TAB)
(right-click - modify on it)
