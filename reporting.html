Crystal - simple distribution of printed / PDF reports. No real interactive query functionality. Note that there are rumours in the press of a possible change in pricing which makes the default client viewer licenced per-user, rather than free viewer.

Brio - can do distribution of "BQY" canned reports with free viewer.  Is sold as being a powerful query analyser tool which works over the web (via plugin) But main strength is the good Javascript model which allows the building of "EIS screens" to give pushbutton / search box / sort etc. without the user having to understand how to use pivottables and drill-down graphs.

Business Objects - more related to standard views which allow certain drillable functionality.

All three are avaiable as web distributed - and all three can send out static reports (to some degree of functionality or another)

Actuate report server for report generation - produces ROI (proprietary format), PDF and spreadsheet in its 5.0 release. Actuate
reports have rich content and can be hyper-linked. Actuate runs on both Unix and NT servers. Example of usage: generate 30,000 reports each night.
